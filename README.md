# 개발환경 가이드북

## 소개

본 가이드는 개발 환경 설정에 어려움을 겪는 초보 개발자를 위해 작성되었습니다. WSL2와 vscode를 사용해  Python3, C / C++, Rust 언어의 기초적인 개발 환경을 설치합니다. _가이드를 따라해 발생한 모든 문제의 책임은 실행한 본인에게 있습니다._
