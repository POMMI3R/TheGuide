# Python3 설정

## Python3 설치

yay를 사용해 `python`과 `python-pip` 패키지를 설치합니다.

```text
yay -S python python-pip
```

## vscode 설정

1. `Python` 확장을 검색해 설치합니다.
2. `py` 확장자 파일을 엽니다.
3. 우측 하단의 경고창을 통해 `pylint`를 설치합니다.
4. `Ctrl` + `Alt` + `n`으로 파일을 실행합니다. 다음의 Python3 예제를 통해 확인하십시오.

```python
print("Hello, world!")
```
